# Retrospective Templates That Don't Suck

6 themed retrospective formats with SVG templates and facilitation guides. Print them, use them digitally, customize them. Each format gives the facilitator a character to play and team members a creative way to reflect.

Read the full article: [article.md](article.md)

## Templates

**[The Newspaper](newspaper/)** - Editor-in-Chief - General-purpose, great for first-timers

**[The Scientific Journal](scientific-journal/)** - Lab Director - Technical teams, data-heavy sprints

**[The Sports Report](sports-report/)** - Commentator - High-momentum, competitive sprints

**[The Late Night Talk Show](talk-show/)** - Show Host - Teams that need to lighten up

**[The Travel Magazine](travel-magazine/)** - Travel Editor - Journey-focused, milestone sprints

**[The Detective Case File](detective-casefile/)** - Chief Inspector - Bug-heavy sprints, post-mortems

## Each folder contains

- `template.svg` - Full template for teams of 6-10 (800x1100)
- `template-short.svg` - Short template for teams of 4-5 (800x780)
- `example-filled.svg` - Filled example showing how it looks when completed
- `FACILITATION_GUIDE.md` - Full script with ice breakers, prompts, and discussion questions

## How to use

1. Pick a template that fits your sprint's vibe
2. Choose the right size (full or short) based on team size
3. **For Miro/Mural:** Import the SVG, **lock it**, then have participants place transparent text blocks on top of each section
4. **For print:** Hand out copies with markers
5. Pre-fill sprint metrics (facilitator) before the session
6. Follow the facilitation guide
7. Let the team name their publication

## Structure (all templates follow the same 55-min timeline)

**Prep (-10 min):** Choose theme, prepare template, pre-fill sprint metrics

**0:00 - 0:05:** Set the stage - open in character, name the publication, explain sections

**0:05 - 0:12:** Ice breakers - collect mood, shoutouts, quote (facilitator fills template)

**0:12 - 0:24:** Individual writing (DIVERGE) - participants fill their sections

**0:24 - 0:29:** Silent gallery walk (CONVERGE)

**0:29 - 0:40:** Headline sharing - round-robin summaries

**0:40 - 0:50:** Group discussion - drop the character, real conversation, action items

**0:50 - 0:55:** Closing - capture action items, close in character

## Three data sources

**Facilitator prepares** (before retro): Sprint metrics, stats, dates - from Sprint Review

**Ice breaker data** (collected together): Mood/weather, kudos/shoutouts, quote of the sprint

**Individual reflections** (participants write): Main story, articles, opinions, classifieds
